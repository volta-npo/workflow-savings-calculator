# Workflow Savings Calculator v1.0 Product Specification

## Workbench

Automation ROI Calculator

## Domain table

Workflow tasks

## Metrics

- Monthly Savings
- Payback Period
- Fragility Risk

## Domain rows

- Manual steps listed
- Frequency entered
- Minutes per run entered
- Error cost estimated
- Build effort estimated
- Maintenance cost estimated
- Risk factors checked
- Recommendation generated

## Generated deliverables

- ROI brief
- Scenario CSV
- Build/no-build memo

## Validation gates

- Frequency/time/cost required
- High-risk workflows cannot auto-greenlight
- Owner approval required
