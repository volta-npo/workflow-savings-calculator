# Contributor Covenant Code of Conduct

## Our pledge

We pledge to make participation in this project welcoming, respectful, and useful for students, mentors, nonprofit operators, small-business owners, and community technologists.

## Expected behavior

- Be kind, specific, and evidence-oriented.
- Assume good intent while protecting users from harm.
- Respect privacy, accessibility, and local context.
- Give actionable feedback that helps contributors learn.

## Unacceptable behavior

Harassment, discriminatory language, doxxing, credential sharing, spam, and unsafe advice are not tolerated.

## Enforcement

Project maintainers may remove comments, close issues, or block contributors whose behavior undermines a safe open-source community.

## Scope

This code applies in issues, pull requests, discussions, documentation, community events, and any official project space.
